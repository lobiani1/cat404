A fun 404 page where a curious cat follows your cursor around the screen, meows when clicked, and sits when you stop moving.

### Features
- Smooth cat movement using JS animation loop
- Idle state with sitting animation
- Meow sound on click :3

### How to Run
1. Clone this repo
2. Open `index.html` in your browser
3. Move your mouse and enjoy the company

### Assets
You can use any pixel cat sprites or small PNGs (e.g. from [OpenGameArt](https://opengameart.org))
